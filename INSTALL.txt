-----------------------------------------------------------
Requirements
-----------------------------------------------------------

Apache version 2
PHP version >= 5.3
MySQL version >= 5.1

Composer (https://getcomposer.org/)

-----------------------------------------------------------
Installation
-----------------------------------------------------------

1. Download project from https://github.com/dasigr/fatwallet and place it in your web direcoty.

i.e.
C:\xampp\htdocs\fatwallet (for XAMPP on Windows)
/var/www/fatwallet (for CentOS)

2. Download required packages

# composer update
